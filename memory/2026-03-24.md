## Portfolio balance formula — confirmed

The `trg_portfolio_update_new` trigger recalculates balance on every trade UPDATE:
```
balance = starting_balance + SUM(pnl WHERE won/lost) - SUM(dollar_size WHERE pending/pending_fill)
```
`expire()` phantom credits are immediately overwritten by this trigger — so the pre-existing `was_deducted=True` bug on between_bucket is harmless. Portfolio tracking is correct.
